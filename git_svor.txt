Git spurningar og svör

1. Hvað gera eftirfarandi Linux skipanir?
	cd
	cd ~
	ls
	pwd
	mkdir

svar:
cd = til að fara í aðra möppu
cd ~ = fer til baka á home möppuna
ls = til að sjá hvaða skrár eru í möppu
pwd = segir í hvaða möppu þú ert staddur
mkdir = býr til nýja möppu

2. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?
	git clone
	git log
	git status
	git diff
	git checkout

svar:
git clone = taka afrit af repository
git log = skoða sögu af git verkefni
git status = segir hvaða breytingar hafa verið gerðar í möppunni sem 	þú ert að vinna í, hvaða breytingar hafa ekki verið gerðar og 		hvaða skrár eru ekki skráðar í Git
git diff = sýnir breytingar milli commita
git checkout = skiptir á milli branches
